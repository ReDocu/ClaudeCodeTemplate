# 99_Archive — 보관소

## 이동 규칙 (모든 이동은 사용자 확인 후 수행)

| 대상 | 조건 | 이동 위치 |
|---|---|---|
| 종료된 프로젝트 폴더 | 프로젝트 handover에 종료 명시 | `teams/{팀명}/{프로젝트명}/` |
| 오래된 결과물 | `11_doc_result`에서 90일 경과 | `doc_result/{YYYY}/` |

- `/report_root` 실행 시 이동 후보를 대시보드의 「아카이브 후보」 섹션에 제안한다.
- 아카이브된 항목은 대시보드·INDEX 본문에서 제거하되, git 이력으로 추적 가능하다.
