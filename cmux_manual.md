# cmux 매뉴얼

> 출처: https://cmux.com/
> macOS용 네이티브 터미널 — 멀티태스킹·조직화·프로그래머빌리티에 특화

---

## 1. cmux란?

cmux는 "멀티태스킹, 조직화, 프로그래머빌리티를 위해 만들어진 터미널"입니다. **Ghostty의 렌더링 엔진(libghostty)** 위에 구축된 **무료 오픈소스 네이티브 macOS 터미널** 애플리케이션으로, 여러 터미널 기반 AI 에이전트와 코딩 워크플로를 관리하는 개발자를 위해 설계되었습니다.

### 해결하는 문제

여러 동시 터미널 프로세스, 특히 AI 코딩 에이전트를 조율하는 어려움을 해결합니다. 숨겨진 백그라운드 프로세스를 **보이고 제어 가능한 패널**로 대체하고, 에이전트가 주의를 필요로 할 때 실시간 알림을 제공합니다.

---

## 2. 주요 기능

- **세로 탭(Vertical tabs)** — 사이드바에 git 브랜치, 작업 디렉터리, 포트, 알림 표시
- **알림 링(Notification rings)** — 패널이 주의를 필요로 할 때 시각적 경고
- **분할 패널** — 탭 내 수평·수직 분할
- **내장 브라우저** — API로 제어하는 스크립트 가능 브라우저 패널(테스트용)
- **프로그래머빌리티** — 자동화를 위한 CLI 및 Unix 소켓 API
- **GPU 가속** — 네이티브 Swift + AppKit 렌더링 (Electron 아님)
- **경량** — 네이티브 macOS 구현
- **오픈소스** — GitHub에 GPL 라이선스로 공개
- **키보드 단축키** — 광범위한 커스터마이징 가능 단축키
- **iOS 컴패니언** — iPhone/iPad로 실시간 터미널 동기화 (베타, TestFlight)
- **세션 지속성** — 재실행 시 창·워크스페이스·패널·스크롤백 복원

---

## 3. 지원 에이전트

명령줄에서 실행 가능한 모든 터미널 기반 코딩 에이전트와 동작합니다:

> Claude Code, Codex, OpenCode, Gemini CLI, Kiro, Aider, Goose, Amp, Cline, Cursor Agent 등

---

## 4. 멀티 에이전트 오케스트레이션

Claude Code 팀 및 oh-my-opencode 멀티 모델 오케스트레이션을 지원하며, 모든 에이전트를 숨겨진 프로세스가 아닌 **네이티브 패널**로 표시합니다.

---

## 5. 원격 기능

- **SSH 워크스페이스** 지원
- **원격 tmux 세션 연결** 지원
- 에이전트를 원격 호스트에서 실행하면서 로컬에서 제어 가능

---

## 6. 브라우저 자동화

cmux는 터미널 옆에 실제 브라우저 패널을 분할할 수 있으며, 동일한 소켓 API를 통해 다음을 지원합니다:

- 프로그래밍 방식 내비게이션
- DOM 스냅샷
- JavaScript 실행
- 콘솔·네트워크 모니터링

---

## 7. 스킬 시스템

**cmux-skills** 리포지토리를 통해 재사용 가능한 워크플로를 제공합니다:

- CLI 제어
- 워크스페이스 자동화
- 설정
- 브라우저 조작

---

## 8. 커스터마이징 및 설정

| 항목 | 위치 / 방법 |
|------|-------------|
| 터미널 설정 | Ghostty 설정에서 읽음 (`~/.config/ghostty/config`) |
| cmux 설정 | `~/.config/cmux/cmux.json` |
| 알림 | 표준 터미널 이스케이프 시퀀스(OSC 9/99/777) 또는 CLI 명령으로 트리거 |

---

## 9. 설치 및 가격

- **설치**: macOS용 다운로드 (`/download/confirmation?dl=1`)
- **가격**: 완전 무료 오픈소스
- **Founders Edition**(선택): 예정 기능(cmux AI, iOS 앱, Cloud VMs) 조기 접근 및 개발 지원을 위한 옵션

---

## 10. 플랫폼 & 주요 사항

- **Ghostty 포크 아님**: libghostty를 렌더링 라이브러리로 사용 (앱에서 WebKit 쓰는 방식과 유사)
- **플랫폼 제한**: 현재 macOS 전용. Linux·Windows·Android는 개발 중 (대기자 명단 있음)
- **iOS 앱**: TestFlight에서 "cmux BETA" 베타 제공
- **tmux 대비**: 터미널 기반 멀티플렉싱 대신 세로 탭·내장 브라우저를 갖춘 GUI 네이티브 경험 제공

---

## 11. 커뮤니티 반응

Ghostty 제작자 Mitchell Hashimoto 등 주요 인물이 지지했으며, 알림 시스템·세로 탭·향상된 멀티태스킹 효율에 대한 호평이 이어지고 있습니다. 다수 사용자가 iTerm2와 Warp를 대체해 주력 터미널로 사용 중이라고 보고합니다.
