# 00_Team — 팀별 작업 공간

팀 폴더는 손으로 만들지 말고 **루트 세션에서 `/new_team {팀명}`으로 생성**하세요.
(표준 CLAUDE.md·handover 문서가 자동 스캐폴딩됩니다.)

## 구조

```
00_Team/
└── {메인폴더명}_{팀명}/          ← 팀 폴더 (예: Template_TeamA)
    ├── CLAUDE.md                 팀 총괄자
    ├── handover_{팀명}.md        팀 인수인계 (하위 프로젝트 취합)
    └── {프로젝트명}/             ← /new_project로 생성
        ├── CLAUDE.md             프로젝트 관리자
        └── handover.md           프로젝트 인수인계
```

## 규칙

- 팀 작업 세션은 **자기 팀 폴더(또는 프로젝트 폴더)에서 시작**한다. 그러면 타 팀·루트 문서에 접근하지 않는 격리가 유지된다.
- 세션 종료 시 `/handover`, 보고가 필요할 때 `/report`를 실행한다.
- 팀명은 영문/숫자/하이픈 권장 (파일명·링크 호환성).
