# CLAUDE.md — 패키지(Package): {팀명}

> 소속 팀: {팀명} · 이 폴더는 패키지 역할의 작업 공간이다.
> 루트·팀 CLAUDE.md의 규칙이 이 폴더에도 적용된다.

## 임무
- 개발 결과물의 빌드·패키징·배포 준비를 수행한다.
- 릴리스 노트 작성, 산출물 검증·정리.

## 권한
- 빌드 스크립트·배포 설정·릴리스 문서를 작성·수정한다. 소스 코드 수정은 개발(`02_developer`)의 몫이다 — 결함 발견 시 handover로 되돌린다.

## 작업 규칙
- 빌드 스크립트·배포 설정·릴리스 문서는 모두 이 폴더 안에 둔다.
- **세션을 마칠 때 `/handover`를 실행**해 `handover_package.md`를 갱신한다.
- 검증 중 발견한 결함은 「다음 할 일」에 남겨 개발로 되돌린다 (직접 수정 금지).
- 상위 폴더(팀 루트·타 역할)의 파일을 직접 수정하지 않는다. 개발 결과물 정보가 필요하면 사용자에게 해당 handover 공유를 요청한다.
