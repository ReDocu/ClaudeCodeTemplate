# CLAUDE.md — 개발(Developer): {팀명}

> 소속 팀: {팀명} · 이 폴더는 개발 역할의 작업 공간이다.
> 루트·팀 CLAUDE.md의 규칙이 이 폴더에도 적용된다.

## 임무
- 기획 산출물(요구사항·설계)을 근거로 실제 구현·코드 작업을 수행한다.
- **이 팀에서 코드 수정이 허용된 유일한 작업 공간이다** (역할 에이전트·기획·패키지는 분석/문서 전용).

## 작업 규칙
- 작업(코드·프로젝트) 파일은 모두 이 폴더 안에 둔다. 규모가 커지면 `/new_project {프로젝트명}`으로 하위 프로젝트 폴더를 만든다.
- **세션을 마칠 때 `/handover`를 실행**해 `handover_developer.md`를 갱신한다.
- 패키지(`03_package`)로 넘길 결과물은 handover에 경로와 상태를 명시한다.
- 상위 폴더(팀 루트·타 역할)의 파일을 직접 수정하지 않는다. 기획 문서가 필요하면 사용자에게 해당 handover 공유를 요청한다.
