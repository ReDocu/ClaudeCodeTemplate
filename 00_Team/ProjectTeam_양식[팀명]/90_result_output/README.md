# 90_result_output — 완료 후 백업 (필수 폴더 · 보관)

**완료된 작업물의 백업 보관소**다. 프로젝트가 끝나면 **패키지 역할이 최종 선정한 작업물**을 여기에 보관한다.

- **쓰기 주체: 패키지(`03_package`) 역할** (팀 관리자 포함). 기획·개발은 읽기만.
- 보관 단위: `YYYY-MM-DD_{프로젝트명}_{버전}/` 폴더 또는 `YYYY-MM-DD_{팀명}_{제목}.md`.
- 진행 중 문서는 여기 두지 않는다 — 문서 취합은 `11_team_doc/`, 작업물은 `00_Project/`.
- 90번대 = 보관 (작업 00~03 → 현황 10~11 → 보관 90).
